# 🌿 Growing Code: Python Fundamentals

A practical project focused on mastering Python 3.10+ core concepts through real-world community garden data scenarios.

## 🎯 Project Overview

This repository contains solutions designed to learn fundamental programming concepts, data manipulation, and software development best practices.

## 🚀 Key Concepts Covered

* **Basic Syntax:** Variables, inputs, and outputs.
* **Control Flow:** Conditional logic and loops.
* **Algorithmic Thinking:** Iterative vs. Recursive approaches.
* **Code Quality:** PEP 8 compliance via `flake8` linting.
* **Type Safety:** Explicit type annotations checked with `mypy`.

## 📂 Repository Structure

The project is organized into modular independent exercises:

* **`ex0/` (Hello Garden):** Basic string display and environment configuration.
* **`ex1/` (Garden Name):** Interactive data input and static messaging.
* **`ex2/` (Plot Area):** Arithmetic computations and basic mathematical calculations.
* **`ex3/` (Harvest Total):** Multi-input gathering and variable accumulation.
* **`ex4/` (Plant Age Check):** Conditional thresholds evaluation.
* **`ex5/` (Water Reminder):** Time-based logic and automation alerts.
* **`ex6/` (Count to Harvest):** Comparative implementation of iteration and recursion.
* **`ex7/` (Seed Inventory):** Advanced data typing, input parsing, and string manipulation.

## 🛠️ Requirements & Standards

* **Runtime:** Python 3.10+
* **Linter:** `flake8` standard
* **Type Checker:** `mypy` (strict verification)
